    Interface ACLChainOps<ChangesType, PublicKey>

    Adapter that the chain uses to talk to a concrete ACL backend.

    The chain is generic over the ACL change block type (ChangesType) and the public key type (PublicKey). It does not itself know how to apply a change block to an ACL state -- it delegates to this adapter.

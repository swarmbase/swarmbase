Interface ACL<ChangesType, PublicKey>

An ACL keeps track of a list of user's public keys and produces changes that can be sent to other swarm peers.

Type parameters

  • ChangesType

    A block of CRDT change(s).

  • PublicKey

    Type of a user's public key.

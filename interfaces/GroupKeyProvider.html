    Interface GroupKeyProvider

    GroupKeyProvider abstracts the group key agreement protocol (BeeKEM or DCGKA).

    It manages a ratchet tree of member keys and produces shared secrets from which epoch encryption keys are derived. Implementations of this interface handle the tree structure, path secret derivation, and key encapsulation internally.

    OptionalkeyID

    keyID?: string

    Identifier for the document encryption key used to encrypt this block. Allows recipients to select the correct decryption key from their keychain without trying all available keys. Preserved through serialize/deserialize round-trips.

    Stored as a string (rather than raw bytes) for JSON serialization safety: Uint8Array values do not survive JSON.parse/JSON.stringify round-trips. Callers should use hex or base64 encoding when creating keyID values from raw byte identifiers.
